# Deferred TUI

Этот каталог содержит прежний `proteus-tui`, вынесенный из активного workspace
на время переезда UI на Leptos.

Статус:

- не входит в root `Cargo.toml`;
- не собирается `./install.sh`;
- не является primary dogfood client;
- сохранён как reference/native prototype для возможного возврата позже.

Historical docs лежат в `docs/` внутри этого каталога: прежний direction
checkpoint, TUI UX research и premortem по старому app-server/TUI риску.
